# Portafolio_Analisis_Datos
Portafolio de proyectos asociados a fundamentos en análisis de datos.

Este repositorio reúne proyectos desarrollados en Google Colab durante mi formación en análisis de datos.  
Incluye ejercicios de limpieza, transformación y visualización de datos usando **Pandas, NumPy y Matplotlib**.

## 📂 Proyectos incluidos
- Sistema Gestión de contactos
- Preparación de Datos
- Análisis exploratorio de Datos
- Inferencia Estadística

## ⚙️ Tecnologías utilizadas
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Google Colab  

## 🚀 Cómo usar
1. Descarga o clona este repositorio.  
2. Abre los notebooks en Google Colab o Jupyter.  
3. Instala las librerías necesarias con:  
   ```bash
   pip install pandas numpy matplotlib seaborn
